# Berber Barber Shop

Premium men's grooming website for **Alaa Bou**, professional barber with 5+ years of expertise.

## Getting Started

To run this project locally, follow these steps:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/project2025gl-ops/berber.git
    cd berber
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

4.  **Build for production**:
    ```bash
    npm run build
    ```

## Features

- **Modern UI**: Dark-themed, premium design with glassmorphism effects.
- **Service List**: Detailed menu of grooming services and pricing.
- **Portfolio**: Showcase of precision cuts and styles.
- **Responsive**: Fully optimized for mobile, tablet, and desktop.
